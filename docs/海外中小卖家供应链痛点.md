## 一、项目背景与市场痛点

全球跨境电商持续增长，海外中小卖家（亚马逊新手、独立站店主、小型品牌创始人）对中国供应链的依赖度极高。但在实际采购过程中，他们长期被以下**四大信息黑洞**困扰：

| 痛点维度 | 具体表现 |
| :--- | :--- |
| **信任危机** | 分不清对方是工厂还是贸易商；担心收了定金就消失；证书、图片PS泛滥。 |
| **质量焦虑** | 大货与样品不一致；不确定是否符合美国Prop 65、欧盟REACH等法规；缺乏真实买家评价。 |
| **沟通与执行风险** | 语言障碍导致细节误解；MOQ过高、交期经常性延迟。 |
| **隐性成本陷阱** | 搞不清FOB与出厂价区别；运费、关税无法提前测算；对跨国维权毫无信心。 |

现有解决方案（阿里巴巴、1688等B2B平台）的底层逻辑是“**尽可能多地展示商品**”，而不是“**尽可能深地验证供应商**”。海外中小卖家急需一个**中立、智能、高效**的工具，将模糊的“找工厂”过程，转变为清晰的“数据驱动决策”。


## 二、产品定位与核心价值

### 产品定位
**“中国供应商的第三方AI体检中心”** ——我们不卖货，不偏向任何供应商，只做一件事：**帮海外买家把中国供应商翻个底朝天，生成一份看得懂、信得过的评估报告。**

### 核心价值主张
> **“3分钟，让一个陌生中国工厂在你面前变得透明。”**

它解决的不是“找到产品”的问题（那是平台的事），而是解决了“**找到之后，敢不敢下单、怎么谈判、怎么避坑**”的核心决策问题。

## 四、核心功能架构（五大模块）

我们将产品划分为**三层服务**，由浅入深，覆盖用户完整决策链：

### 第一层：信息聚合层（输入）
- **多平台搜索入口**：用户输入产品关键词、图片或1688/阿里链接，系统自动抓取该商品及对应店铺的公开数据。
- **数据来源**：1688、阿里巴巴国际站、中国制造网、国家企业信用信息公示系统、海关进出口数据（如ImportYeti风格）。

### 第二层：AI智能评估层（核心大脑）
这是项目技术壁垒所在。AI将聚合的数据清洗、交叉验证后，输出一个**综合信任评分（100分制）**，并拆解为五个子维度的结构化报告：

| 评估维度 | 具体输出字段（精简版） |
| :--- | :--- |
| **① 企业身份核验** | 工商全称、注册年限、**工厂/贸易商判别指数（%）、** 实控人关联风险、参保人数。 |
| **② 真实交易信誉** | 近6个月成交笔数、**复购率（回头率）、** 主要出口目的国、平台纠纷率（脱敏）。 |
| **③ 产品硬实力** | 主营产品匹配度、**有效期内认证证书（CE/FCC/REACH等）、产线实拍AI真实性评分**。 |
| **④ 商业条款评估** | 最低起订量（MOQ）区间、**历史实际交货平均天数（非承诺值）、** FOB/EXW状态标识、阶梯价格表。 |
| **⑤ 风险雷达** | 知识产权侵权司法记录、制裁名单筛查、经营异常预警。 |

### 第三层：决策转化层（输出）
- **多语言报告生成**：一键切换英语、西班牙语、法语等，彻底消除语言障碍。
- **AI行动建议（点睛之笔）**：直接给用户生成**3条英文询盘话术模板**。
  - *示例*：“这家自称有BSCI，但证书是2022年的，建议你直接问：‘请提供2025年最新的BSCI审核报告全文和验厂照片。’”
- **红黄绿灯可视化**：
  - 🔴 红色（风险项，一票否决）
  - 🟡 黄色（预警项，建议追问）
  - 🟢 绿色（通过项，值得信赖）

| **实体经营年限** | 公司注册年限 + 当前场地租赁合同（若公开）。 | 判断抗风险能力，规避骗完就跑的新壳公司。 |
| **实际控制人关联图谱** | 该公司法人是否同时关联多家同类公司（若公开）。 | 识别专门用来“接投诉、弃壳”的风险主体。 |

### 模块二：📊 真实交易信誉（回答“有没有人买过，靠不靠谱”）

基于平台交易数据与海关记录，替代“虚假好评”。

| 返回字段 | 具体说明 | 解决痛点 |
| :--- | :--- | :--- |
| **近6个月真实成交笔数** | 聚合1688/阿里国际站的实际动销数据（非刷单量）。 | 识别“有价无市”的虚假繁荣。 |
| **复购率（回头率）** | 该商家有多少比例的客户在持续下单。 | **核心指标**：复购率高说明质量稳定，而非一锤子买卖。 |
| **主要出口目的国（海关数据）** | 如果该供应商有海关记录，显示其货主要发往美国、欧洲还是东南亚。 | 如果目标市场有该品类，说明匹配度高；若从未出口过，需警惕包装/标签经验不足。 |
| **平台纠纷率（脱敏）** | 历史交易中关于“质量不符”、“交期延迟”的投诉比例。 | 直接量化“大货与样品不符”的风险。 |

### 模块三：🧪 产品硬实力与合规（回答“东西行不行，能不能卖”）

解决质量和法律合规问题，这是中小卖家最怕踩的雷。

| 返回字段 | 具体说明 | 解决痛点 |
| :--- | :--- | :--- |
| **主营产品线匹配度** | AI提取该工厂近一年上架/拍摄的SKU，判断是否为“专业老厂”还是“倒手二道贩子”。 | 避免万能杂货铺类型的低质贸易商。 |
| **行业认证检测报告（有效期）** | 提取如CE、FCC、RoHS、REACH、FDA证书编号及**到期日**。 | **命中美国Prop 65和欧盟法规焦虑**。需特别标注“已过期”或“临期”。 |
| **产线实拍AI真实性分析** | 对商家上传的工厂图片进行AI水印、背景重复度分析，标注“疑似网图/渲染图/实拍”。 | 帮你判断“虚假实力展示”。 |
| **质量管控体系** | 是否具备ISO9001，或是否提到来料检验、过程检验、出厂检验（IQC/IPQC/OQC）。 | 预判大货质量稳定性。 |

### 模块四：💰 商业条款与隐性成本（回答“钱怎么算，多久到”）

计算到岸成本（Landed Cost），这是利润生命线。

| 返回字段 | 具体说明 | 解决痛点 |
| :--- | :--- | :--- |
| **最低起订量（MOQ）区间** | AI提取该链接或店铺展示的最低订货门槛（如100pcs），并标记“是否支持混批/试单”。 | 解决小卖家“资金不足”的痛点。 |
| **历史交货周期（天）** | 基于用户评价或平台发货数据，推算**实际平均交货天数**（非承诺天数）。 | 解决“交期延迟错过销售季”的焦虑。 |
| **报价含税含运费状态** | 明确标注报价是“FOB（离岸价）”还是“出厂价（EXW）”，并**提供预估海运费/空运费参考链接**。 | 帮用户快速算总账。 |
| **起定量阶梯价格表** | AI整理该产品在不同数量级（100/500/1000pcs）下的历史成交单价。 | 帮助用户做库存与利润的量化决策。 |

### 模块五：⚠️ 风险雷达与AI行动建议（回答“该不该聊，怎么聊”）

这是你产品的**护城河**，直接把复杂数据转化为傻瓜式指令。

| 返回字段 | 具体说明 | 解决痛点 |
| :--- | :--- | :--- |
| **司法风险提示** | 是否有过**知识产权侵权**（商标/专利）官司？是否被列入经营异常名录？ | 防范“设计被盗用”或“连坐封店”风险。 |
| **制裁/黑名单筛查** | 是否出现在美国实体清单（Entity List）或其他制裁名单（公开数据）。 | 确保账号安全，避免亚马逊关联封号。 |
| **AI智能评分（100分制）** | 综合以上所有维度，给出一个**客观的推荐分**（例如：82/100，值得接触）。 | 减少用户选择困难。 |
| **个性化话术建议** | **这是点睛之笔**。AI生成3个针对该供应商的**英文/本地语言提问模板**。<br>例如：“这家工厂自称有BSCI认证，但证书显示是2022年的，建议你直接问：'请提供最新的2025年BSCI审核报告全文'。” | 让买家在谈判中显得“懂行”，提升议价成功率。 |


### 🖥️ 前端页面呈现建议

不要把20个字段平铺下来。建议采用**红黄绿灯**可视化：

- 🔴 **红色风险（一票否决项）**：如经营异常、严重侵权纠纷、刚成立不到1年。
- 🟡 **黄色预警（需追问项）**：如证书快过期、无海关出口记录、评价数量极少。
- 🟢 **绿色通过（可信赖项）**：如复购率高、实拍认证通过、成立超5年。

**核心原则**：你提供的不是“信息”，而是“**决策依据**”。你每多提供一个量化的交叉验证数据，用户对你产品的付费意愿就增加一分。
