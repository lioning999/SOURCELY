# 1688 卖家等级数据差异

> 目录索引：[docs/README.md](README.md)
> 生命周期：活跃 | 创建：2026-07-22

---

## 一句话结论

1688 不同卖家等级（sellerType）使用**不同的数据字段**存储相同语义的数据。`_map_raw()` 只读了一种路径，漏掉了 `supplier.factoryTags` 中的数据。

---

## 两种数据存储路径

### 路径 A：`stats` 直接字段 + `rawCardDetail`（yuantou_flagship 专用）

```json
"stats": {
  "positiveReviewRate": 0.993,
  "repeatRate": 0.766,
  "repeatBuyers": "100+",
  "crossBorderBuyers": "80+",
  "rawCardDetail": [
    {"code": "goodRate", "title": "好评率", "info": "99.3", "unit": "%"},
    {"code": "byrRepeatRate", "title": "回头率", "info": "76.6", "unit": "%"}
  ]
}
```

### 路径 B：`supplier.factoryTags`（super_factory 专用）

```json
"supplier": {
  "factoryTags": [
    {"text": "回头率", "value": "63%"},
    {"text": "履约率", "value": "100%"},
    {"text": "服务分", "value": "5.0"}
  ]
}
```

### 路径 C：无数据（normal / normal_factory）

两个位置都为空。1688 平台对普通商家不展示此类数据。

## 字段映射修复方案

`_map_raw()` 当前只读 `stats`：

```python
# 当前（只读一条路径）
"shop_rate":  _parse_pct(stats.get("positiveReviewRate")),
"repurchase": _parse_pct(stats.get("repeatRate")),
```

需要改为**多路径 fallback 链**：

```python
# 修复后（stats → rawCardDetail → factoryTags）
def _from_factory_tags(factory_tags, key):
    for item in (factory_tags or []):
        if item.get("text") == key:
            return item.get("value")
    return None

"shop_rate":  (_parse_pct(stats.get("positiveReviewRate"))
               or _parse_pct(_from_card_detail(stats, "goodRate"))
               or _parse_pct(_from_factory_tags(d.get("supplier", {}).get("factoryTags"), "好评率"))),

"repurchase": (_parse_pct(stats.get("repeatRate"))
               or _parse_pct(_from_card_detail(stats, "byrRepeatRate"))
               or _parse_pct(_from_factory_tags(d.get("supplier", {}).get("factoryTags"), "回头率"))),
```

### 新增字段

`factoryTags` 中的 `履约率` 和 `服务分` 是当前数据模型没有的字段，可以考虑：
- `履约率` → 工厂可靠性指标，可入判词
- `服务分` → 服务态度指标，可入判词

是否新增由产品决定（MVP 阶段不做也行）。

## 对前端的影响

| 卖家等级 | 展示效果 |
|---------|---------|
| yuantou_flagship | 完整：好评率 + 回头率 + 认证 + 排名 + 买家数据 |
| super_factory | 中等：回头率（来自 factoryTags）+ 认证 + 排名（可能为空） |
| normal_factory | 基础：shop_years + 工厂标签 |
| normal | 最基础：shop_years + 标签（免费拿样等） |

首页三张样例卡刚好覆盖了三种数据完整度，是合理的产品设计。
