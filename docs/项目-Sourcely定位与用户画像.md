## 一、目的是什么

**帮海外卖家跨越 1688 信息壁垒，回答一个核心问题：这个品值不值得进货。**

Sourcely 的本质是**"1688 采购决策引擎"**。

| 不是 | 是 |
|------|-----|
| ❌ 1688 数据翻译器 | ✅ 采购决策辅助工具 |
| ❌ 下单代理 | ✅ 工厂可信度判断引擎 |
| ❌ 信息展示页 | ✅ 从"好奇"到"拿样"的转化漏斗入口 |

样品代采是用户做出"我要试这个品"决策之后自然发生的交易——不是产品的核心价值主张。

### 一句话差异化

> 别的代购说"你发链接我帮你买"，Sourcely 说"我先告诉你这个工厂能不能合作，再帮你买。"

Sourcely 卖的不是代购，是**判断力**。1688 上一个手机壳有 5000 家店，海外卖家不知道哪家靠谱。替他判断，这才是核心价值。

## 二、客户是谁

**东南亚电商卖家**，在 Shopee / TikTok Shop / Lazada 上开店，需要从中国进货。

### 三种使用场景（占比为推测）

| 用户类型 | 占比 | 场景 | 核心需求 | 对 report 页面的期望 |
|----------|------|------|---------|---------------------|
| 🔍 **比价调研型** | ~60% | 看到竞品爆款，想知道进货价多少、利润空间多大 | 30 秒内知道"进货价 × 零售价 → 利润空间" | "一看价格就走" |
| 🏭 **找货源型** | ~25% | 有明确品类要做，在找靠谱的 1688 工厂供应商 | 验证工厂是否可靠——认证、年限、销量 | "告诉我能不能合作" |
| 📦 **拿样型** | ~15% | 已看中一个品，想知道拿样流程和费用 | 知道"多少钱、怎么操作、多久到" | "告诉我怎么操作" |

### 典型用户画像

**卖家 A — 小规模试水型**
- 越南 · Shopee · 配件/百货 · 月销 ~200 单
- 核心恐惧：差评扣分（Shopee 对差评极其敏感）
- 价格敏感度：高
- 一句话："我先拿 2-3 个样品试试，质量 OK 再小批量。别让我一次订 500 件。"

**卖家 B — 中等规模增长型**
- 印尼 · TikTok Shop · 美妆/个护 · 月销 ~800 单
- 核心恐惧：爆单断货（TikTok 算法推起来就 3 天的事）
- 价格敏感度：中（宁愿多花 20% 确保供应链稳定）
- 一句话："断货比贵更致命。"

**卖家 C — 稳定经营品质型**
- 泰国 · Lazada · 家居/饰品 · 月销 ~300 单
- 核心恐惧：货不对板（之前被代购坑过，图片 A 级品、发货 B 级品）
- 价格敏感度：中低（验货不是成本，是保险）
- 一句话："没验货我不下单。照片确认了再发国际。"

## 三、report 页面扮演什么角色

**转化漏斗第一层。** 用户搜 1688 链接 → 得到工厂分析 → 产生信任 → 想要拿样 → 咨询 WhatsApp。

```
好奇 → 搜索 → 分析结果 → 信任建立 → 拿样决策 → WhatsApp 咨询
                          ↑
                    report 页面在此
```

如果用户得到分析结果后信任没建立，漏斗就断了。

### 当前核心矛盾

页面目前按**"拿样型用户"**设计（价格计算器、两段付款、服务承诺、Chat on WhatsApp），但 **~60% 的用户只是来比价的**。比价型用户看到这些会产生"我还没决定要进这个品，为什么一直在催我下单"的压力。

**页面需要首先服务 60% 的比价用户——让他快速得到结论——然后自然地引导他去拿样。**

### report 页面的三个功能层次

| 层次 | 功能 | 服务谁 |
|------|------|--------|
| **第一层：快速结论** | 判词（推荐/谨慎/避开）+ 价格 + 利润空间感知 | 比价型（60%） |
| **第二层：工厂可信度** | 认证、年限、销量、数据完整度 | 找货源型（25%） |
| **第三层：拿样操作** | 费用计算、两段付款、服务承诺、WhatsApp | 拿样型（15%） |

三个层次的权重应该递减，用户从第一层往下自然流动。

## 四、竞品位置

| 竞品 | 做法 | Sourcely 的优势 |
|------|------|----------------|
| 个人代购（微信群/Facebook Group） | "你发链接我帮你买" | 多了"工厂判断"能力，不是纯跑腿 |
| 1688 跨境专供频道 | 1688 官方跨境专区，有英文 | 多了第三方中立判断，不只翻译 |
| 速卖通 AliExpress | 面向海外零售 | 价格比速卖通低很多，是"源头价" |
| 自己翻 1688 + Chrome 翻译 | 手动找货 | 省 2 小时 + 多了工厂可信度分析 |

**最直接的竞争对手不是某个产品，而是"自己翻 1688"这个行为。** Sourcely 赢在"判断力"和"时间节省"，不是价格。
