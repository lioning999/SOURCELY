## 数据可用等级说明

| 等级 | 含义 | 判定依据 |
|------|------|---------|
| **A** | 100% 必返（6/6 全有） | 不管什么 sellerType/flags 都返回 |
| **B** | 旗舰独有 | `flags.isYuantouFlagship == True` |
| **C** | 工厂级有 | `isYuantouFlagship \|\| isSuperFactory \|\| isFactory` |
| **D** | 不稳定 | 同类卖家也可能没有，取决于 1688 商品本身 |

## 七、当前已知缺陷

1. **回头率漏取** — `super_factory` 的回头率在 `supplier.factoryTags` 中，`_map_raw()` 未读取，导致木盒子/项链的 trust-bar 中回头率显示为空
2. **好评率 = 旗舰专属** — normal / normal_factory / 纯 super_factory 都无法获取，1688 平台本身不返回
3. **排名不稳定** — 同是 super_factory，木盒子有二钻工厂、项链没有，非代码问题
4. **认证 = 工厂专属** — normal 贸易商没有认证数据
