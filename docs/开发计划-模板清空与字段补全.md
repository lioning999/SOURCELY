## 文件 1：`src/api/services/analyze_svc.py`

### 1a. 修阶梯价格键名

- **根因：** 读 `priceRanges` / `priceRange`，Apify 实际键名是 `quantityPrices`
- **改动：** 一行补 `quantityPrices` 为优先键名

### 1b. `_map_raw()` 补 6 个字段

| 字段 | Apify 路径 | 映射后的 key |
|------|-----------|-------------|
| 商家身份 | `supplier.sellerType` | `sellerType` + `sellerTypeLabel`（转中文） |
| 认证类型 | `supplier.certification.type` | `certType` + `certReportUrl` |
| 品类排名 | `supplier.rank.text` | `rankText` |
| 工厂标签 | `supplier.flags` | `factoryFlags`（解析为中文标签数组） |
| 商品标签 | `productFlags` + `serviceLabels` | `badgeLabels`（中文标签数组） |
| 回头客/跨境买家 | `supplier.stats` | `repeatBuyers` / `crossBorderBuyers` |

sellerType 中文映射：`yuantou_flagship→源头旗舰`、`shili_factory→实力工厂` 等
flags 中文映射：`isFactory→生产厂家`、`isShiliFactory→实力工厂` 等
